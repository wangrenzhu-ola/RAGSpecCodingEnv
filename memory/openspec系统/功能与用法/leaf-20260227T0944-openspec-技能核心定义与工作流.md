- OpenSpec 是规格驱动开发（SDD）的核心工具，用于锁定需求、防止范围蔓延
- 适用场景：新功能、行为变更、重构、架构变更或模糊需求
- 工作流：Proposal (提案确认) -> Apply (实施开发) -> Archive (归档验收)
- 核心命令：npx openspec proposal/apply/archive/changes
